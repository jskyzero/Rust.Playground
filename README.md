## Tips

+ install: `curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`
+ offiline docs: `rustup docs`, `rustup docs --book`
+ single file: `rustc helloworld.rs`
+ new project: `cargo new playground`
